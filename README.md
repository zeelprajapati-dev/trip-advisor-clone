# ✈️ Trip Advisor-AI (Clone)

## 📌 Project Overview

**Trip Advisor-AI** is a full-stack web application built on the **MERN Stack** (MongoDB, Express.js, React, Node.js) that simplifies travel planning. The platform integrates an **AI Agent** (JavaScript-based) to provide interactive and personalized travel recommendations.

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| 🔐 **Authentication** | User registration & secure login |
| 🎙️ **AI Chat/Call Assistant** | Real-time AI help for travel queries |
| 📝 **Trip Planning** | Create, manage, and customize trips |
| 📜 **Trip History** | View all past trips |
| 💳 **Secure Booking** | Safe payment mechanism |
| ⭐ **Feedback System** | Rate and review your experience |

## 🛠️ Tech Stack

- **Frontend:** React, HTML/CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **AI Integration:** Custom JavaScript AI Agent
